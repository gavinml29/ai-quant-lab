Stock Indicator Lab — 技术指标分析报告

版本：v7 | 日期：2026-07-04 | 数据：7 只 A 股，3 年日线 (2023-07 ~ 2026-07)

股票池：三安光电、金风科技、铂力特、信维通信、绿的谐波、兆易创新、北京君正

1. 工具概述

index.html 是一个纯静态 Web 应用（双击即可打开），基于 ECharts 5.5 渲染，内嵌 7 只 A 股的 3 年 OHLCV 数据。提供以下交互功能：

  • 股票切换：左侧下拉选择器，切换后全部图表实时重绘
  • K 线图：原生 candlestick + MA5/10/20/60 + 成交量柱状图，含 dataZoom 缩放/平移
  • RSI 面板：参数 N 可调 (2~100)，含 30/70 参考线
  • MACD 面板：FAST/SLOW/SIGNAL 三参数可调，含金叉/死叉统计
  • BOLL 面板：N 和 K 可调 (K 支持 0.1 精度)，含突破/跌破计数
  • ATR 面板：参数 N 可调，含波动率百分比显示
  • Reset 按钮：一键恢复所有默认参数

2. RSI — 相对强弱指标

计算公式

RSI = 100 − 100 / (1 + RS)
RS = avg_gain / avg_loss
Wilder 平滑: avg_t = (avg_{t-1} * (N-1) + value_t) / N

参数 N 的影响分析

N 值灵敏度波动范围适用场景
5~7 (超短)极高，频繁触及 30/7020~80超短线交易 (1~3 日)，日内波动捕捉
14 (默认)适中，< 5次/年触极值30~70中短线波段交易，经典 Welles Wilder 推荐
21~28 (中期)较低，曲线平滑35~65中期趋势确认，过滤短线噪音
50+ (长线)极低，接近水平线40~60长线持仓参考，不适合波段操作

交易信号指导

  • 买入 RSI < 30 且开始回升（超卖反弹）：配合 N=14 使用，确认信号需等 RSI 上穿 30 线
  • 卖出 RSI > 70 且开始回落（超买回调）：不建议仅凭 RSI 做空，应结合趋势判断
  • 背离 价格新高但 RSI 未创新高（顶背离）→ 趋势衰竭，减仓信号
  • N 值调小：信号更频繁但假信号增多，适合高波动品种（如 688333 铂力特）
  • N 值调大：信号更可靠但滞后，适合低波动蓝筹（如 600900 长江电力）

3. MACD — 指数平滑异同移动平均线

计算公式

EMA(N)_t = close_t * 2/(N+1) + EMA(N)_{t-1} * (1 − 2/(N+1))
DIFF = EMA(FAST) − EMA(SLOW)
DEA = EMA(SIGNAL) of DIFF
MACD 柱 = 2 * (DIFF − DEA)

参数组合影响分析

参数组合金叉频率滞后性适用场景
(12,26,9) 默认约 15~20 次/3年2~3 根 K 线标准中短线，适合大多数股票
(6,13,4) 灵敏型约 40~60 次/3年1 根 K 线超短线或高波动品种，假信号多
(24,52,18) 稳健型约 5~10 次/3年5~8 根 K 线中长线趋势跟踪，适合蓝筹
(5,34,5) 异类FAST << SLOW严重滞后长线投资参考，忽略短期波动

交易信号指导

  • 金叉 (Golden Cross) DIFF 上穿 DEA → 买入信号。默认参数下金叉出现在趋势启动初期，但需结合零轴位置：零轴上方金叉强于零轴下方
  • 死叉 (Dead Cross) DIFF 下穿 DEA → 卖出信号。零轴上方死叉可视为回调，零轴下方死叉是趋势转弱确认
  • 零轴判断 DIFF > 0 为多头市场，DIFF < 0 为空头市场。MACD 柱由绿转红是短线转强信号
  • SIGNAL 参数：调大 SIGNAL (如 18) 使 DEA 更平滑，减少假信号但滞后增加；调小 SIGNAL (如 5) 信号更及时但噪音多
  • FAST/SLOW 差距：差距越大（如 FAST=5, SLOW=50），DIFF 振幅越大，适合趋势分明的品种；差距越小，DIFF 接近零轴，信号微弱

4. BOLL — 布林带

计算公式

MIDDLE = MA(close, N)
STD = σ(close, N)
UPPER = MIDDLE + K * STD
LOWER = MIDDLE - K * STD

参数影响分析

参数效果交易含义
N=10, K=2带宽窄，突破频繁适合短线波段，假突破多
N=20, K=2 (默认)带宽适中，突破偶发标准配置，突破信号较可靠
N=50, K=2带宽宽，极少突破中长线趋势通道，突破意义重大
N=20, K=1.5带窄 25%，突破增加适合震荡市，高抛低吸
N=20, K=3带宽 50%，极少突破适合趋势市，突破是强信号

交易信号指导

  • 突破上轨 收盘价 > UPPER → 强势突破信号。但在震荡市中可能是假突破，需配合成交量放大确认
  • 跌破下轨 收盘价 < LOWER → 弱势信号。若连续多日在下轨运行，表明趋势转弱，应减仓
  • 布林带宽度 (UPPER − LOWER)：宽度收缩到极小值时，预示即将变盘（突破上轨或下轨）
  • 中轨方向：中轨向上且价格在中轨上方 → 多头趋势；中轨向下且价格在中轨下方 → 空头趋势
  • K 值调节策略：高波动股票（如科创板 688 开头）可适当扩大 K 值至 2.5~3，减少误报；低波动股票可缩小至 1.5~2，提高灵敏度

5. ATR — 平均真实波幅

计算公式

TR = max(High − Low, |High − Close_{t-1}|, |Low − Close_{t-1}|)
ATR = Wilder 平滑(TR, N)
波动率% = ATR / Close * 100

参数 N 的影响分析

N 值ATR 特征用途
5~7 (超短)反应极快，波动剧烈日内止损设置、短线仓位管理
14 (默认)平滑适中，反映近期平均波幅标准止损距离、波动率监控
21~28 (中期)非常平滑，忽略短期冲击中长线仓位管理、趋势判断
50+ (长线)极慢，反映长期波动水平长期风控参考、资产配置

交易信号指导

  • ATR 本身不产生买卖信号，是波动率度量工具，用于辅助仓位管理和止损设置
  • 止损距离：建议止损设在入场价 ± 1.5~2 倍 ATR。A 股日线 ATR 通常在 0.5~3 元之间，应根据个股实际 ATR 值动态调整
  • 仓位管理：ATR 增大时应减仓（风险增加），ATR 缩小时可适当加仓（风险降低）
  • 波动率突破：ATR 突然飙升（>2倍均值）往往伴随重大消息或趋势转变，应警惕
  • 横盘识别：ATR 持续走低 → 横盘整理阶段，不宜重仓参与
  • 趋势确认：上升趋势中 ATR 温和放大是健康信号；下跌趋势中 ATR 放大需警惕加速下跌
  • 跨股票比较：使用 ATR/Close% 而非绝对 ATR，才能公平对比不同价位股票的波动性

6. 多指标共振策略

组合信号强度操作建议
RSI > 50 + DIFF > DEA + 价格在 BOLL 中轨上方 + ATR 正常 强多头 4 指标一致偏多，可积极持仓或加仓
RSI < 30 + DIFF < DEA + 价格跌破 BOLL 下轨 + ATR 放大 强空头 4 指标一致偏空，应减仓/止损
2 多 2 空 分歧 市场方向不明，建议观望或轻仓
RSI 顶背离 + MACD 死叉 + 突破 BOLL 上轨后回落 顶部确认 高概率见顶，建议大幅减仓
RSI 底背离 + MACD 金叉 + 跌破 BOLL 下轨后回升 底部确认 高概率见底，可逐步建仓

7. 工具使用说明

  1. 双击 index.html 在浏览器中打开
  2. 选择股票：K 线图左侧下拉框选择目标股票
  3. 调节参数：每个指标面板左侧滑块拖动可实时看到曲线变化
  4. 对比分析：所有图表同宽，可将 K 线图与指标图上下对齐对比
  5. 重置：每个面板有独立 Reset 按钮，顶部有 Reset All
  6. 数据更新：修改 data/ 目录下 CSV 后，运行 python3 build.py 重建 index.html
